## How to Run Locally
1. Front-end Setup
    - Install dependencies:
      ```bash
       npm install
      ```
    - Run the frontend:
      ```bash
      npm start
      ```
2. Back-end Setup
    - Create a virtual environment:
      ```bash
        python -m venv venv
      ```
    - Activate the virtual environment:
    - ```bash
      venv\Scripts\activate
      ```
    - Install dependencies:
    - ```bash
      pip install -r requirements.txt
      ```
    - Run the backend:
    - ```bash
      flask run
      ```
